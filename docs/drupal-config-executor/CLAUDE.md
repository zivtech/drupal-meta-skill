# drupal-config-executor

`drupal-config-executor` consumes planner output and generates Drupal config YAML artifacts.

## Command

- `/drupal-config-executor`

## Role

- Upstream: `drupal-planner.content-model`, `drupal-planner.taxonomy`, `drupal-planner.search`
- Downstream: `drupal-critic` and domain critics such as `content-model-critic`
